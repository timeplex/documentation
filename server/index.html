Usage of plexi interface in contiki

plexi scheduler interface is a CoAP-based mechanism that exposes node resources, e.g. TSCH slotframes and links, to external schedulers. The first implementation of plexi interface is done on Contiki OS v3.0. That is, it relies on the coap rest engine present in that version of Contiki. It is also assumed that RPL and/or IEEE802.15.4e-TSCH is enabled in the node.
